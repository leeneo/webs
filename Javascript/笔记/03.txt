Js第三天
一、	条件判断语句
1、	单项判断
If（判断条件）{ 执行的代码 }
解释：如果判断条件是真的，才执行大括号中的代码
2、	双向判断
If(判断条件){ 代码段1 }else{ 代码段2 }
		解释：如果条件是真的，就执行代码段1，否则就执行代码段2
3、	多项判断语句
If(条件1){ 如果条件1是真才执行代码段1 }else if(条件2){如果条件2是真才执行代码段2  }else if(){}…..else{ 对比之前的条件都不满足 才执行这里的代码 }
注意：如果是判断范围的值，要从大到小来写。
二、	逻辑运算符
1、或 ||   1真1假 结果真
2、且 &&  1真1假 结果假
3、非 ！   取反，如果是true的非就是false，反之就是true,叹号写在之前。
三、for循环
	for(var I = 0; I < 99; i++){
		此处写需要重复执行的代码
}
解释：循环的三个重要的量：初始值、循环次数、循环的增量
注意：分号只有两个。
xxx.length  获取元素的个数信息。
当群体控制的时候才要使用for循环。
三、	标签获取页面元素
Var aDivs = Document.getElementsByTagName(‘div’)
注意：aDivs此时是一个多元素变量，不能直接控制。只能通过中括号加数字的形式来获取第几个元素，js从0开始计数。
aDivs[0]  获取第1个div元素

四、	数组
数组可以理解为变量组，它的符号是[]。
数组也可以使用.length来获取数组中元素的个数。
五、	随机数公式
Math.random() 可以生产0到1之间的小数
如果想生成时整数，就要使用parseInt()来去掉小数点（不是四舍五入）
parseInt(Math.random()  * 5)    0、1、2、3、4
