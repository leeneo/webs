Js第二天
一、	Js书写位置
1、	行内（强烈建议不使用）
<button onclick=”alert(‘heihei’);” >按钮</button>
注意：在代码中引号是以就近原则成对解析的。
2、	内嵌（课上使用）
放在script标签中使用
3、	外链（最常用的）
Script标签设置src属性，把外部的js代码引入到页面中
<script src=”js.js”>   注意：此处如果再写代码是无效的  </script>
二、	js搬家
1、	window.onload=function(){  此处再写js代码  }
我们今后工作中都是把js代码写在head标签中，这样就务必在代码的最外层添加一个加载事件才能实现效果。此加载事件也叫“入口函数”。
三、	js中的this
在js中，事件源就是this。点谁谁就可以用this这个单词来替换
	注意：用单引号引起来的会原样输出，不加单引号js会计算后再输出。

四、	五大数据类型
1、	数值型 number
例： 1  0.2    421
2、	字符串型 string
例：’1’  ‘asd’  ‘哈哈’   
注意：字符串类型如果相加，只是简单的连接在一起，并没有执行相加操作。
3、	布尔型  Boolean
例： true  false   
4、	对象型
默认值： null
5、	未定义

五、	检测、转化
检测数据类型：typeof()  想检测谁，就把谁放在小括号中
转化：parseInt()  想转化谁，就把谁放在小括号中
六、	加号
两个数字相加才是数学上的相加，其它的时候用加号代表把两个数据连接起来，我们叫它连字符。
七、	自定义函数
Function 名字(){
	要执行的代码
}
调用:  函数名+小括号  函数如果不调用，那么里面的代码永远都不会被执行。
小括号中可以写参数，js会偷偷的声明变量来保存这个参数。

八、	自定义函数书写步骤
如果发现重复性的代码段，我们就要考虑使用自定义函数来简化它。
1、	先写自定义函数
2、	把重复的代码段复制到大括号中
3、	在大括号之外调用自定义函数（函数名+小括号）
4、	分析不同点，然后在调用的时候写参数。
5、	变量名可以修改成更合理的。
