Flexible Box Layout（简称 Flex 布局） 详解

  • 在 CSS3 以前，CSS2.1 的时候，定义了4种布局方式：
    • block layout(块布局)，用于文档排版
    • inline layout(内联布局)，用于文本排版
    • table layout(表格布局)，用于表格形式的2维数据排版
    • positioned layout(定位布局)，用于非常精确的元素定位，不会在意其他元素
  • Flex layout(弹性布局) 用于更为复杂的应用和网页排版
  • Flex Layout Box Model :弹性盒模型，CSS3 中新出现的一种盒模型
  • Flex 布局就是以这种 FlexBox 盒模型来进行页面排版的一种布局方式，准确的讲应该叫 Flex Container
  • A flex container is the box generated by an element with a computed display of flex or inline-flex.
    In-flow children of a flex container are called flex items and are laid out using the flex layout model.
  • 此片段参考自：https://www.w3.org/TR/css-flexbox/#intro

  • 核心概念：flex layout is biased to the flex directions

  • 抽象理解就是采用水平轴(官方叫 main axis)/垂直轴(官方叫 cross axis)分别对应水平，垂直方向
  • 水平轴是主要的布局轴，弹性元素以此方向为基准来进行排版
  • 水平轴与边框的左侧交界线和右侧交界线，分别叫做 main start 和 main end
  • 垂直轴与边框的顶部交界线和底部交界线，分别叫做 cross start 和 cross end
  • main start 与 main end 的左右间距就是 main size 也就是水平轴长
  • cross start 与 cross end 的上下间距就是 cross size 也就是垂直轴长
  • 此片段参考自：https://www.w3.org/TR/css-flexbox/#box-model

  • 所谓 flex 布局，就是以此 container 内的水平，垂直方向，大小（水平宽度，垂直长度）为蓝图， 定义 container 以及其中的 items 的大小，位置，伸缩，对齐，间距等等排版方式
