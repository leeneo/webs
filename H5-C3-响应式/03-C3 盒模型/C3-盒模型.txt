一、	新增颜色模式
在 C3中提供了一种 rgba() 来表达颜色的语法，里面可以设置四个参数，前三个分别表示 r g b 的数值，最后一个参数表示透明度，小数点前的0可以省略。
只要是能用 颜色表达的地方我们都可以使用  rgba 来替代【文字颜色，边框颜色，背景色......】
Background:rgba( r,g,b,alpha )

二、	背景尺寸
Background-size: 
注意：
01 利用该属性可以用来设置背景图片的显示尺寸
02 设置具体的数值则表示背景图片的宽高，但是设置一个数值时仅表示宽度，二个参数之间用空隔隔开。
03 设置百分比同样表示宽高，百分比计算的基准值是相对于容器的宽高来计算的。
04 cover关键字，一般用来将小图进行放大可以铺满整个容器。规则就是按着原图的比例来缩放当前图片，有可能会超出容器。
05 contain关键字，一般用来将大图缩小在一个容器中，也是按着比例来缩放图片，当某一边到达容器的显示范围之后就会停止缩放，有可以会有留白。

三、	内减模盒型
按着标准盒模型来说，如果我们在 css 中给某个元素定义了一个 width 属性值为 200px ，
那么如果后面继续添加 padding border ，它就会在200px 的基础上继续累加得到最终显示的盒子大小。
而设置 box-sizing:border-box 的元素如果设置了 border  padding ，那么它会在 200px 的基础上来减去这一部分的值，得到最终显示大小【200px】。

四、	过渡属性
Transition：

注意：
01 transition 属性可以在当前元素的某些 CSS 属性值发生变化的时候产生一个过程。
02 第一个参数一般用 all 表示所有属性都过渡，也可以单独的去设置某一个属性过渡。
03 第二个参数是用来设置当前过渡需要的时长
04 第三个参数是用来设置过渡的动画形式，C3中默认提供了 ease linear ease-in ease-in-out ease-out
05 最后一个参数可以用来设置当前过渡延时多久使用，单位必须是 s 即使为0也要加上这个单位。


五、	2D转换属性
在 C3中提供了 2D 和 3D 二类转换，但是目前来说使用相对较多是 2D平面内的转换。不论是哪种形式的转换，都是设置 transform 属性
1.	位移
Transform: translate(  )
注意：
01 小括号中可以设置具体的数值也可认设置百分比。
02 设置一个参数的时候只表示 X 轴，二个参数之间用逗号隔开。
03 设置百分比的时候是相对于元素的宽高来做为基准值进行计算的
04 默认情况下转换的起始位置是相对于元素的中心点来计算的，可以为正也可以为负。

2.	缩放
Transform: scale( 系数 )
注意：
利用 scale 可以对当前元素进行缩小或者放大的操作，它改变的是元素的宽度和高度。
1 在它后面的小括号中设置具体的数值，表示的是缩放的系数（倍数）。这个值可以是大于1 也可以是小于1。
2 如果只设置一个参数那么和 translate 不同的地方就在于它表示 宽和高二个值都进行缩放
3 如果设置二个参数那么中间用逗号隔开。分别表示宽 高

3.	旋转
Transform:rotate( 角度 )
注意：
利用该属性可以让元在平面范围内旋转一定的角度，面对屏幕顺时针为正，反方向为负。角度单位是 deg .
01 只设置一个参数的时候我们研究的就是在平面内围绕着垂直屏幕一个轴做旋转。

4.	斜切
Transform:skew( 角度 )

注意：
01 skew 属性目的是为了让当前元素显示出一些透视的3D关系。因为我们现在主要还是在2D平台内展示页面所以该属性不常用。
02 小括号中设置一个参数表示X 轴方向角度，设置二个参数中间用逗号隔开，分别表示 水平和垂直。
03 角度值可以为正也可以为负
